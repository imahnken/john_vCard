A) HTML Structure - top

This theme is a single page personal site, similar to the vcard based sites made popular by Tim Van Damme. The template is divided in two main areas (DIVs) - left and right. The left side contains the navigation elements and the right side contains all the content in an unordered list, which is contained inside a DIV container for the jQuery sliding effect to work. Here's the broad HTML document structure.

I've included two more color variations of the template in addition to the default one, so there are in all three colors available - red (default), blue and green. To choose a different color, just change the stylesheet reference in the head section from style.css to style-blue.css or style-green.css. You don't need to edit the HTML to change colors. Just edit the background image, and change some color values in the CSS file. See the CSS section below for details on creating color variations.

Content for each of the four section is contained within an li in the container DIV. Here's a look at how the "about me" section is structured. I've followed similar logic for the rest, so you should be able to follow along fine.

B) CSS Files and Structure - top

All styles for the content are in the style.css file. The CSS is structured like this:

	/* --- RESET --- */

	An adaptation of Eric Meyer's CSS reset for cross-browser consistency

	/* --- GLOBAL TYPOGRAPHY --- */
	
	All primary text styling
	
	/* --- STRUCTURE --- */
	
	Document structure, layout and positioning
	
	/* --- NAVIGATION --- */
	
	Styling for the navigation menu
	
	/* --- CONTENT - XYZ --- */
	
	Styling specific to each of the four content sections

There are two other CSS file - style-blue.css and style-green.css - which are for the two color variations. There are two things different in all these CSS files:

  1. The background image for the body tag
  2. Color values for the h2 tag and links (to change these, simply find and replace #f30 from the style.css to the hex code of whatever color you want to use.

Please note that this template uses the CSS opacity property, which is not supported by Internet Explorer. As a result, the navigation links will not change opacity when clicked in IE.

C) JavaScript - top

This template imports three Javascript files:

  1. jQuery: Used for the rollover and content sliding effects, as well as to be able to call specific selectors for Cufon font replacement.
  2. Cufon: For using the custom font "Harabara" in the navigation and social links
  3. Harabara_700: The font definition created using Cufon.

It also uses custom jquery definitions for:

  • replacing the fonts for the navigation and social site links using Cufon,
  • defining the rollover and opacity change on the navigation links, and
  • defining the sliding animation of content blocks.

D) PSD Files - top

Included with the template are 4 PSD files:

  1. strokes.psd: The base PSD file which contains all the content layouts and site elements.
  2. strokes_bg.psd: Source file for the background image.
  3. strokes_bd-blue.psd and strokes_bg-green.psd: Source files for color variations of the background image.

To create more color variations, just change the gradient in the layer styles for the navigation background layer.

E) PHP Files - top

The file sendmail.php contains the necessary code for the contact form to work. To make sure the messages from the contact form go to you, change the e-mail ID in line 7 of this file to your e-mail ID. Note that this functionality requires a PHP-enabled server and will therefore only work on a web server. It will not work off of your desktop.
